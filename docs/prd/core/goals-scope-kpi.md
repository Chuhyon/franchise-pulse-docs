# PRD Core: Goals, Scope, KPI

## 데이터 소스 전략 (요약)

- `data.go.kr`: 장기 기본 소스
- `localdata.go.kr`: 전환기 보조 소스
- `KFTC`: 브랜드/가맹 메타데이터 보강
- 지자체 오픈데이터: 공백/지연 구간 보완

## In Scope (v1)

- 대시보드: 월별 개업/폐업 Top 100
- 필터: 월, 지역, 업종, 브랜드 검색
- 브랜드 상세: 월별 추이 및 순증감
- CSV 내보내기
- 품질 표시: 마지막 동기화 시각, 신뢰도 배지

## Out of Scope (v1)

- 사용자 계정/구독
- 분 단위 실시간 수집
- 예측 모델/추천 엔진

## 성공 지표

- 데이터 최신성: 일 단위 갱신 성공률 95% 이상
- 파이프라인 신뢰성: 스케줄 작업 성공률 99% 이상
- 사용자 효용성: 필터 사용 세션 비율 40% 이상
- 정확성: 샘플 대사 검증 오차 1% 미만
- 가용성: 퍼블릭 대시보드 월 가용성 99.0% 이상
