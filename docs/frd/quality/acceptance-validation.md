# FRD Quality: Acceptance & Validation

## 인수 기준 (AC)

- AC-01: 특정 월 선택 시 개업/폐업 Top 100이 3초 이내 렌더링
- AC-02: 일 수집 작업이 랭킹과 최신성 시각 갱신
- AC-03: 지역/업종 필터 결과가 결정론적으로 일관
- AC-04: CSV 결과가 화면 테이블 정렬과 일치
- AC-05: 소스 상태 페이지가 어댑터별 마지막 정상 동기화 표시
- AC-06: 샘플 대사 검증 시 월 집계 오차 1% 미만
- AC-07: 단일 소스 장애 시 degraded 배지 + 마지막 정상 집계 제공

## 데이터 품질 검증 규칙

- 중복 검증: 소스 키 + 소스 갱신시각 조합 유일성
- Null 검증: `event_date`, `event_type`, `source_record_id` 누락 금지
- 범위 검증: 비정상 미래 날짜 이벤트 차단
- 분류 검증: 미분류 비율 2% 초과 시 경보
